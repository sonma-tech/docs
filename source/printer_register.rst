打印机注册
------------

.. _mfg:

设置打印状态
^^^^^^^^^^^^

接口地址
~~~~~~~~~~~~~~

.. http:get:: /printer/mfg
.. http:post:: /printer/mfg

请求参数
~~~~~~~~

============ =============== ========================================================================
参数名称       是否必须          说明
============ =============== ========================================================================
sn                 是               打印机唯一编号
============ =============== ========================================================================

返回示例
~~~~~~~~

.. code-block:: json

    {
        "sn": 2000708988,
        "type": "GPZH816",
        "online": false,
        "status": 0,
        "organization": {
            "id": 1,
            "name": "滚马",
            "device_online": 0,
            "device_total": 45
        }
    }

.. _generate:

生成打印机信息
^^^^^^^^^^^^^^^^^^^^

接口地址
~~~~~~~~~~~~~~~~~

.. http:get:: /printer/generate
.. http:post:: /printer/generate

请求参数
~~~~~~~~~~~~~~~~~

============ =============== ========================================================================
参数名称       是否必须          说明
============ =============== ========================================================================
group              是               打印机区间
remark             是               打印信息
type               否               打印方式，默认GPZH816
============ =============== ========================================================================

返回示例
~~~~~~~~~~~~~~~~~

.. code-block:: json

    {
        "sn": 2000708988,
        "type": "GPZH816",
        "remark": "测试打印机",
        "online": false,
        "status": 0,
        "key": "SsCimP7w"
    }
